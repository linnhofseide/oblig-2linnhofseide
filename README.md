Kort beskrivelse av applikasjon (5–10 setninger): du kan kjøpe kinobilletter med denne appen. Den lagrer informasjon om kunden, både personlig informasjon som navn og telefonnr, og upersonlig informasjon om hva som ble kjøpt. Denne informasjonen kan du se etter at et kjøp har blitt gjort, men den blir også lagret i en array. Du kan også velge å slette innholdet. Det sletter også innholdet i arrayet.
